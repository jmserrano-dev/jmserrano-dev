Software development engineer with more than **10+ years** of experience developing software products in multiple industries (**bookings, public administration, eSignature, insurance, broadcasting services**) in companies that range from established multinationals ([Plain Concepts](https://www.plainconcepts.com/), [Deloitte](https://www2.deloitte.com/)) to start-ups ([Evicertia](https://www.evicertia.com/))
